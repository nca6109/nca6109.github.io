About Me

I am a software developer and game designer with a B.S. in Game Desgin and Development from the Rochester Institute of Technology. I am both a front and backend developer proficient in C#, C++, Javascript, and HTML/CSS and with experience working on websites, databases, and .NET applications. Additionally, I am an experienced game designer from my work on a variety of game projects and years of analyzing the variety of games I play. I also compose, produce, and integrate the soundtracks and effects for a lot of my games and projects. Aside from games and technology, I enjoy playing volleyball, being involved in music, and obsessing over whatever fictional world has piqued my intrigue recently (Star Wars and Attack on Titan are some favorites of mine). I am looking forward to a long career of software development and design and wherever that path may take me.

Gallery

Captive's Call

I worked with a team of four others on a casual puzzle game based on Lawrence Watt-Evans' Legends of Ethshar novel series. Our game, Captive's Call is a tile based puzzle game built in Unity where the player can control up to four unique characters that all have a unique set of abilities. The player can use these abilities to traverse and manipulate the environment in levels to reach the exit. I was responsible for the majority of the gameplay and UI implementation in the game, the sound design and implementation, as well as creating some of the assets. Link to itch.io page: https://pictoparade.itch.io/captives-call

Par for the Curse

I worked with a team of four others on a 3D mini-golf/visual novel game based on Jim Butcher's Dresden Files novel series. Par for the Curse is a mini-golf game where the player completes character themed mini-golf courses in order to solve the mystery of a curse. The game also contains a visual novel-esque hub room where the game's characters can be interacted with. I was responsible for all hub room controls, the dialogue system and some writing, and the sound design and implementation. Link to itch.io page: https://pictoparade.itch.io/par-for-the-curse

Crowd Comic

"Crowd Comic" is an interactive, user created web comic. I worked on Crowd Comic as an development intern. The team from my internship conceptualized and built the project during the summer of 2024. The site was developed with NextJS and the database backend with PostgreSQL and SequelizeJS. Over the course of the project I helped design the overall site's flow, design and build the database, implement API functionality into the front-end, and build the user functionality and services. The repository for Crowd Comic can be found here and the deployed site here

Xenoblade Audio Visualizer

I created an audio visualizer web app using music from and graphic's inspired by Monolithsoft's Xenoblade Chronicles. The visualizer has four different tracks that it can react to, a set of controls that can change how the app visualizes the music or changes the music itself, and an interactive canvas interface. While creating this web app I gained more experience working with JS Modules, drawing with Canvas, and audio anylysers. Link to the visualizer: https://nca6109.github.io/xenoblade-audiovisualizer/

A Webpage of Ice and Fire

I made a website that leverages An API of Ice and Fire to present an interactive map to the user as well as multiple different search pages. HTML/CSS and Javascript were used to make the page and communicate with the API. My intent in creating this project was to gain experience working with external web services and find interesting and interactive ways to provide users information. Link to Website: https://nca6109.github.io/thrones-site/

Golden Guns

Golden Guns is a web based game that uses the PIXI JS rendering engine and javascript. I focused on creating a game that was simple in concept but could be eleborated on to make a complex experience. I wanted to get experience creating a game that interfaces directly with a browser and its local storage capacities to allow for progress to be saved between play sessions. Link to Website: https://nca6109.github.io/golden-guns/

Postmortem Vows

Postmortem Vows is a single player horror-comedy puzzle solving game where the player solves puzzles to progress the story along. I worked in a team with four others over the course of a semester to design and implement this game. I primarily focused on implementiong the item and inventory systems, traversal, and dynamic dialogue boxes after heavy design focus in pre-production.
